## 基于nodejs的用于开发调试的服务端程序

开发人员在调试app的时候可以启动本地环境的服务器，而不用后端人员协助，便于前后端分离

- `npm run test`

 启动本地测试环境

- `npm run proxy`

 在本地启动服务器代理并修改header支持跨域
